About this website

이 퍼즐위치찾기 웹사이트는 말그대로 퍼즐의 위치를 찾아주기 위해서 제작되었습니다. 자세한 사용방법은 서비스의 내용을 참고하여 주십시오.

Services we offer

이 퍼즐위치 찾기 웹사이트는 Django, html, css, js를 기반으로 만들어졌으며, 퍼즐의 위치를 찾는데에는 python과 opencv모듈이 사용되었습니다.

  • 먼저 웹사이트에 접속합니다.
  • 페이지에서 시작버튼을 눌러서 나온 웹캠으로 퍼즐사진을 찍습니다.
  • 그렇게 찍은 사진들중 자신이 원하는 사진을 선택하여 업로드합니다.
  • 그러면, 자동으로 분석이 완료되어 결과사진이 보여집니다.

details

총 서버 기동 -> 웹캠으로 사진 찍기 -> 원하는 사진을 고르기 -> 이미지 크기조정 -> 이미지 색상으로 유사도 디텍팅 -> canny detection을 이용해 edge detecting후 퍼즐의 모양으로 디텍팅 -> 각 결과사진 저장 -> 결과사진들 웹페이지에 출력 의 과정을 거쳐 해당 웹사이트가 기동됩니다.
